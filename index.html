Welcome to my portfolio!

I am a web developer with a passion for programming currently based in Birmingham, England. I enjoy building applications and solving problems.

Since graduating from University i have built three applications, feel free to play with them and checkout the code on git.



Memories Photo App


Contribution: Solo Developer

Description: A photo upload application that allows users to create an account and store photos of precious memories to remember them by.

Under the Hood: For the backend i used Node.JS with express for its simplicity and easy of use, MYSQL to connect to database and abstract information and multer to store uploaded data.
On the front end i used EJS templates to make passing information from the backend easier and more efficient along with some basic JQuery.

Source code on GIT


Word Finder for Scrabble


Contribution: Solo developer

Description: After playing a game of Scrabble for the first time i found myself and my family continously spending time searching through a dictionary to confirm the validation of a word. This inspired me to built a word finder app to identify if a word does exist in the dictionary and if so, what it means.

Under the Hood: On the backend i used Node.JS with MYSQl to look into the database. Then i used websockets to communicate with the server and frontend to pass data in real time.

Source code on GIT
